httpYac

Quickly and easily send REST, SOAP, GraphQL and gRPC requests

Get Started

Feature Rich

Create and execute any REST, SOAP, GraphQL and gRPC queries from within VS Code or CLI

Extensible

The plugin system allows the community to build and share reusable solutions to common needs.

Manage Authentication

httpYac supports many authentication out of the box (OAuth2, AWS, Basic, Digest, SSL Client Certificates)

Variables

Built in support for variables and environments (dotenv, intellij variable support).

Node JS Scripting Support

enrich requests with custom scripts like custom authentication or test scripts

Automated Test

Automate manual tests and integrate them into your CI/CD pipeline

# Getting Started

Install CLI:

npm install -g httpyac
# OR
yarn global add httpyac

Install VS Code Extension

open in new window

code --install-extension anweber.vscode-httpyac

Install VS Code Notebook Extension

open in new window

code --install-extension anweber.httpbook
