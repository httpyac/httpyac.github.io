httpBook ​

httpBook extension provides a Notebook Editor Interface for Http Requests. Notebooks allows you to create and share documents that contain live code, equations, visualizations and narrative text. A detailed description can be found at the VS Code documentation

Installation ​

httpBook can be easily downloaded from the

bash
code --install-extension anweber.httpbook
